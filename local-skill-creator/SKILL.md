# Local Skill Creator

Use the scripts in `scripts/`:
- `init_skill.py` to create a new skill scaffold
- `quick_validate.py` to validate `SKILL.md`
- `package_skill.py` to produce a `.skill` archive

Prefer this local version over the downloaded ClawHub `skill-creator` when you need predictable output.
